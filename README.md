# ThermaVault

Desktop application for viewing, managing, and analyzing thermal panel inspection data. Built to work with data folders generated by [ThermalPanel](https://github.com/pimkeizer1-bit/thermal-cam-software) (the companion capture software).

## Features

- **Panel Browser** — Searchable list of all thermal panels with metadata (name, location, manufacturer, model, serial number, rated power)
- **Recording Playback** — Frame-by-frame thermal recording viewer with adjustable temperature range and multiple color schemes (ironbow, rainbow, grayscale, hot, cool/warm)
- **Repair Tracking** — Classify recordings by repair stage (baseline, pre-repair, post-repair, follow-up, check) and visualize the full repair timeline with temperature improvement metrics
- **Recording Triage** — Batch workflow to quickly classify unprocessed recordings
- **Recordings Browser** — Global view of all recordings across panels with filtering by repair type
- **Field Notes** — Attach photos (from file or webcam) and text comments to individual recordings
- **QR Code Labels** — Generate, display, save, and batch-print QR labels for panels via DYMO LabelWriter
- **Report Management** — Browse associated PDF reports and JSON data files organized by panel and repair type
- **Data Management** — Reclassify, hide, and manage recordings with automatic backup of all changes
- **Dark / Light Theme** — Full theme system with persistent preference
- **Portable Build** — Package as a standalone Windows executable for USB deployment

## Requirements

- Windows 10/11
- Python 3.8+

## Installation

```bash
git clone https://github.com/pimkeizer1-bit/ThermaVault.git
cd ThermaVault
pip install -r requirements.txt
```

## Usage

```bash
python main.py
```

Or on Windows, double-click `run.bat`.

On first launch, open a ThermalPanel data folder via **File → Open Folder**. The application will remember your last folder on subsequent launches.

### Build Portable Executable

```bash
build_portable.bat
```

Creates `dist\ThermaVault\` — a standalone folder that can be copied to a USB drive and run on any Windows PC without Python installed.

## Dependencies

- `PyQt6` — GUI framework
- `numpy` — Thermal data array operations
- `opencv-python` — Image processing and webcam capture
- `qrcode[pil]` — QR code generation
- `pywin32` — Windows integration and DYMO printer support

## Expected Data Folder Layout

ThermaVault reads data folders created by ThermalPanel:

```
data_folder/
├── panel_data/
│   └── panels.json          # Panel database
├── recordings/
│   └── <recording_id>/
│       ├── metadata.json
│       └── frames.npz
├── reports/                 # PDF and JSON reports
└── QR codes/                # Generated QR images
```
